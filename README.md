## Download

[⬇️ Download the latest version of Iris](https://github.com/KeyvanSchneider/Iris-Releases/releases/latest)

Compatible with macOS 13 or later, on Intel and Apple Silicon Macs.

## Features

* Movies, TV shows, and books
* Star and half-star ratings
* Comments and reviews with attachments
* Custom playlists
* Letterboxd import
* Covers and banners via TMDB, Open Library, and ISBNdb
* Light and dark mode
* French and English
* Data stored locally
* Automatic updates

## Installation

1. Download `Iris.zip`.
2. Unzip the archive.
3. Move `Iris.app` to Applications.
4. On first launch, right-click Iris and select **Open**.

## Configure Catalogs

### Movies and TV Shows — TMDB

1. Create an account on [TMDB](https://www.themoviedb.org/signup).
2. Open **Profile → Settings → API**.
3. Request an API key and choose **Developer** usage.
4. Copy the **API Key (v3 auth)** value.
5. In Iris, open **Settings → Movies & TV Shows — TMDB**.
6. Paste the key, then click **Save**.

> Never share your API key publicly.

### Books — Open Library

Open Library works directly in Iris, for free and without an API key. Simply select **Open Library** as your preferred catalog.

### Books — ISBNdb

1. Create an account on [ISBNdb](https://isbndb.com).
2. Choose an available plan.
3. Retrieve your key from your ISBNdb dashboard.
4. In Iris, open **Settings → Books**.
5. Select **ISBNdb**, paste the key, then save.

The keys are stored in the secure macOS Keychain.


## Privacy

Iris runs locally. Your library, comments, and attachments stay on your Mac.
